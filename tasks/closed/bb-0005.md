### Floating action bar
Like how Figma has it we'd like to move using a floating action menu in the bottom of screen, 48px high. For now, we'll only have insert button on it. Clicking insert button, it will open a popup panel just above like Figma does. The popup will have a search bar where users can search a component and components are rendered in grids per the text query. Initially it only show recent components. When user starts typing we filter *all* components per the query and show them. The popup can scroll vertically if there are more components than the popup can fit.
